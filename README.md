# Arinova Packages

Official SDKs, CLI, and plugins for [Arinova Chat](https://chat.arinova.ai).

| Package | npm | Description |
|---------|-----|-------------|
| [@arinova-ai/agent-sdk](packages/agent-sdk) | [![npm](https://img.shields.io/npm/v/@arinova-ai/agent-sdk)](https://www.npmjs.com/package/@arinova-ai/agent-sdk) | Agent SDK for building AI agents |
| [@arinova-ai/theme-sdk](packages/theme-sdk) | [![npm](https://img.shields.io/npm/v/@arinova-ai/theme-sdk)](https://www.npmjs.com/package/@arinova-ai/theme-sdk) | Theme SDK for Office themes |
| [@arinova-ai/spaces-sdk](packages/spaces-sdk) | [![npm](https://img.shields.io/npm/v/@arinova-ai/spaces-sdk)](https://www.npmjs.com/package/@arinova-ai/spaces-sdk) | Spaces SDK for interactive games |
| [@arinova-ai/cli](packages/arinova-cli) | [![npm](https://img.shields.io/npm/v/@arinova-ai/cli)](https://www.npmjs.com/package/@arinova-ai/cli) | CLI for managing messages, notes, kanban, and more |
| [@arinova-ai/openclaw-arinova-ai](packages/openclaw-plugin) | [![npm](https://img.shields.io/npm/v/@arinova-ai/openclaw-arinova-ai)](https://www.npmjs.com/package/@arinova-ai/openclaw-arinova-ai) | OpenClaw plugin for Arinova integration |
